# نبذة تعريفية عن الموقع

هذا الموقع يهدف إلى تسهيل الوصول إلى المواد التعليمية لطلاب الكليات، حيث يوفر محتوى متنوع يشمل:

- فيديوهات تعليمية على يوتيوب
- ملفات ومذكرات على جوجل درايف
- بوستات تعليمية مبسطة

## طبيعة المحتوى

المحتوى المتوفر يركز على شرح المواد الدراسية بشكل مبسط وسهل، ويغطي مختلف الفرق الدراسية.
وهذا الشرح معد من قبل طلاب لديهم خبرة في المواد الدراسية، مما يجعل المحتوى موثوقًا ومفيدًا للطلاب.

## طريقة الاستخدام

1. يقوم الطالب باختيار الفرقة الدراسية الخاصة به.
2. ستظهر له المواد المتاحة لتلك الفرقة مع روابط المحتوى.
3. لا يحتاج الطلاب إلى تسجيل الدخول للاستفادة من المحتوى.

> **ملاحظة:** تسجيل الدخول متاح فقط للمديرين والمساهمين.
